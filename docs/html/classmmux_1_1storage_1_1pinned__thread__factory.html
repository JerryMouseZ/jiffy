enum mmux::storage::pinned_thread_factory::PRIORITY

POSIX Thread scheduler relative priorities,

Absolute priority is determined by scheduler policy and OS. This enumeration specifies relative priorities such that one can specify a priority within a giving scheduler policy without knowing the absolute value of the priority.
